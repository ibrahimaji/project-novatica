### How to use

```bash
npm install 
```

```bash
npm run dev 
```

### How to contribute

1. Create issue
2. Create branch
3. Git checkout to branch `git checkout <branch-name>`
4. Make changes
5. Git add and commit `git add .` `git commit -m "message"`
6. Git push `git push origin <branch-name`
7. Create pull request
